About this map

What the map shows

Due to the inconsistencies in the data available between states we have tried to limit what we displayed to what is common across data sets. We have not shown Application boundaries as they are not necessarily inidicative of actual mining activity. Map data was downloaded between Feb 22 and Mar 5 2013 and is assumed to be current for that time period.

  1. Rock Art Areas: Includes ?
  2. Mines: Includes all Operating Mines and Major Resource Projects (a mining project containing several mines). State level data was used where available, otherwise National Data was used.
  3. Pipelines: Includes Petroleum, Liquids, and Gas pipelines based on available State level data.
  4. Leases and Licences: Includes exploration licences, tenements, claims, leases and licenses based on available State level data.
